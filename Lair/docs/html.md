## HTML
HTML significa literalmente **HyperText Markup Language**, a linguagem de marcação padrão para criar páginas na web. Sua função é ser a estrutura da página e comunicar ao navegador como o conteúdo deve ser exibido para o usuário. "Saber" HTML, na minha opinião, consiste em 3 habilidades: Saber usar as principais tags e sempre escolher uma estrutura **semântica**, entender os **parâmetros** das principais tags, e saber construir uma estrutura **escalável**, **organizada** e que segue **boas práticas**.

## <a id="estrutura-basica"></a> Estrutura Básica
Todo arquivo HTML contém `<!DOCTYPE>`, `<html></html>`, `<head></head>` e `<body></body>`. Essas tags compõe a estrutura básica de um website, eis o que cada uma significa:

### `<!DOCTYPE>`
A declaração DOCTYPE aparece no topo do sua página, antes de qualquer tag. Ela serve para definir o tipo de arquivo e ajudar o navegador a exibir o conteúdo do site corretamente. No caso dos arquivos `.html`, essa declaração fica assim:
```html
<!DOCTYPE html>
<html>
    [...]
</html>
```
