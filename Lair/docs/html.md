## HTML
HTML significa literalmente **Hyper Text Markup Language**, a linguagem de marcação padrão para criar páginas na web. Sua função é ser a estrutura da página e comunicar ao navegador como o conteúdo deve ser exibido para o usuário. "Saber" HTML, na minha opinião, consiste em 3 habilidades: Saber usar as principais tags e sempre escolher uma estrutura **semântica**, entender os **parâmetros** das principais tags, e saber construir uma estrutura **escalável**, **organizada** e que segue **boas práticas**.
